## Styling Details

```
Background: gradient(muted/40 â†’ muted/20)

```

### Thinking Box (Expanded)
```
Same as above, plus:
Steps Container:
  - Padding: 16px (sides), 12px (bottom)
  - Gap: 8px between steps
  
Each Step:
  - Bullet: 4px circle, ai-accent color
  - Text: xs size, muted-foreground
  - Animation: 300ms fade-in + slide-left
```

### Streamed Content (Full Width)
```
Background: transparent (no bubble)
Width: 100%
Text: foreground color
Layout: Natural text flow (like ChatGPT)
Padding: none
Border: none
```

### Typing Cursor

### Data Cards (Streaming)
```

  - Duration: 300ms
  - Stagger: 50ms per card
```

## Responsive Behavior

- **Mobile (< 640px)**: Full screen, thinking box slightly smaller padding
- **Tablet (640px - 1024px)**: Max width 85% for messages
- **Desktop (> 1024px)**: Max width 85% for messages, optimal reading width

## Dark Mode Adaptations

All streaming components automatically adapt:
- Thinking box: Uses theme-aware muted colors
- Content bubble: Gradient adjusts to dark background
- Cursor: Always uses ai-accent (visible in both themes)
- Text: Uses theme foreground/muted-foreground variables
- Borders: Uses theme border variable with opacity

## Accessibility

- Thinking box is keyboard navigable (Enter/Space to toggle)
- Screen readers announce "Thinking process" for the details element
- Typing cursor is visual only (not read by screen readers)
- All interactive elements have focus states
- Color contrast meets WCAG AA standards
